# ISA: Interface between Hardware and Software

## 6 Great Ideas in Computer Architecture

* **Design for Moore's Law** * **Use Abstraction to Simplify Design**
* **Make the Common Case Fast** (Performance Measurement & Improvement)
* **Performance via Parallelism**
    * Instruction-level Parallelism (ILP)
    * Thread-level Parallelism (TLP)
    * Data-level Parallelism (DLP)
    * *Note: Consider **Amdahl's Law** when evaluating speedup.*
* **Performance via Pipelining**
* **Performance via Prediction**
* **Hierarchy of Memories** (Principle of Locality)
* **Dependability via Redundancy**
